# Информационная система управления магазином обуви

## Основной функционал

Программа обеспечивает автоматизацию работы сотрудников магазина (Администратор, Менеджер) и предоставляет интерфейс для просмотра каталога Гостям.

### Возможности:
* **Авторизация:** Разграничение прав доступа (Администратор, Менеджер, Гость).
* **Управление каталогом:**
    * Просмотр списка товаров с динамической подгрузкой данных.
    * Поиск по наименованию, описанию и артикулу.
    * Фильтрация по производителям и сортировка по количеству на складе.
* **Работа с товарами (Admin):**
    * Добавление и редактирование через форму с валидацией данных.
    * Удаление товаров: С проверкой на наличие товара в активных заказах.
* **Работа с изображениями:**
    * Автоматическое изменение размера фото до 300x200.
    * Хранение путей в БД, автоматическая очистка папки `data/` от старых изображений при их замене.

## Стек технологий
* **Язык:** Python 3.1x
* **Интерфейс:** PyQt6
* **База данных:** SQLite3
* **Библиотеки:** `os`, `shutil`, `random`, `string`

## Структура проекта
* `main.py` — Точка входа, логика авторизации.
* `product_list.py` — Главное окно со списком товаров и фильтрами.
* `editor_window.py` — Окно редактирования/добавления товара.
* `shoe_store.db` — Файл базы данных.
* `data/` — Директория для хранения изображений товаров.
